Installation Instructions

Read the entire Test Suite Readme prior to beginning the installation.

Prerequisites
This release of the Test Suite requires either the Java Development Kit (JDK) or the Java Runtime Environment (JRE) from either of the following Java versions:
  • Java SE 6, Update 1 or later
The latest version of this software is available for download free-of-charge at http://java.sun.com/javase/downloads/index.jsp. Archived versions are available at http://java.sun.com/products/archive/. Refer to this site for a list of specific Java versions verified.
Constraints
In addition to the preceding Java prerequisites, the Test Suite requires HTML and JavaScriptTM.

As of this release, the Test Suite was tested on Microsoft® Windows® Vista SP1 and XP SP3 on the Intel Pentium Platform using Microsoft Internet Explorer 7 and Internet Explorer 8.

The Test Suite has not been tested formally with any other browsers on any other platforms than those indicated above.

Installation

The Test Suite is available for download on the ADLNet.gov Web site as a self-extracting executable install file. After you download the file:

  1. Double-click the downloaded self-extracting executable file to begin the installation process.
  2. Click Next to continue from the "Welcome" page.
  3. Read the "License Agreement" and
    • Click Yes to agree with the terms of the license and continue with the install.
    • Click No if you do not agree with the terms of the license and want to exit the install.
  4. The recommended root installation directory is "C:\ADL". You may choose another root installation directory, but the Test Suite install will always append the "\SCORM_2004_4th_Ed_TS_V1.1.1" sub-directory to the specified root installation directory. The remainder of this Readme document assumes that the Test Suite will be installed to "C:\ADL\SCORM_2004_4th_Ed_TS_V1.1.1". This is referred to as "<install_directory>". If you override the default to use a different root installation directory, you must account for that when reading the remainder of this Readme.
  5. Click Next.
  6. The install will add program icons to the Program Folder of your Start menu. Type a new folder name, or select one from the existing list of Start Menu folders. "ADL" is the recommended folder. Click Next to install the Test Suite and the installation program will copy the necessary files to the appropriate directories.
  7. Click Finish to complete the installation.

The installation process adds a shortcut for running the Test Suite to your desktop. It also adds the following shortcuts to the "Programs" folder of your Start menu:

  • Link to ADLNet.gov
  • Shortcut to this Readme document
  • Shortcut to the Test Suite

The installation will not remove or un-install previous versions of the SCORM Test Suite. If you want to remove a previous version:

  1. Open the Windows "Control Panel".
  2. Open "Add or Remove Programs".
  3. Highlight the "SCORM Test Suite <version>".
  4. Click Change/Remove; and follow the instructions.
The LMS Test Content Packages are not included in the install file. These files are needed to run the LMS Conformance Test. You can download the LMS Test Content Packages separately at the ADLNet.gov Web site.

Configuration

To ensure that the Test Suite runs properly after it is installed:

  • The Internet browser must be set to check for a new page every visit to the page. To configure the browser:
    • For Microsoft Internet Explorer 6 Service Pack 2
      1. Open a Microsoft Internet Explorer browser.
      2. Click "Tools" then "Internet Options".
      3. On the "General" tab, within the "Temporary Internet Files" sub-section, click Settings.
      4. Ensure that the "Every visit to the page" radio button is selected.
      5. Click OK and then OK again.

    • For Microsoft Internet Explorer 7 and 8
      1. Open a Microsoft Internet Explorer browser.
      2. Click "Tools" then "Internet Options".
      3. On the "General" tab, within the "Browsing history" sub-section, click Settings.
      4. Ensure that the "Every time I visit the webpage" radio button is selected.
      5. Click OK and then OK again.

  • The Java Plug-in must be configured to disable caching. To disable the Java Plug-in cache:
    • For Java SE 6
      1. Open the Windows "Control Panel".
      2. Double-click the "Java" icon.
      3. Select the "General" tab.
      4. Under the "Temporary Internet Files" section, click Settings.
      5. Ensure the "Keep temporary files on my computer." checkbox is unchecked.

  • Ensure that the Sun Java Plug-in is selected as the default Java Environment for Internet Explorer. To set the default browser Java Environment:
    1. Open the "Java Plug-in" applet in the Control Panel.
    2. Select the "Browser" tab.
    3. Select the checkbox next to the browser that you are using to run the Test Suite.
    4. Click Apply.

  • If you are using Microsoft® Windows® XP SP3 these additional actions may be required:
    • Allow JavaScript to run by:
      1. Opening an Internet Explorer browser.
      2. Clicking "Tools" then "Internet Options".
      3. Check the box that states "Allow active content to run in files on My Computer" on the "Advanced" tab, under the "Security" section.
    • Allow the Test Suite to pop-up windows by either disabling pop-up blocking or excluding the Test Suite from pop-up blocking.

This completes the installation of the Test Suite software. Review the "Running the Conformance Tests" and "Known Issues" sections of this Readme prior to running the Test Suite. This Readme is installed to the installation folder specified in step #4 above. (e.g. "<install_directory>\SCORM_2004_4th_Ed_TS_V1.1.1\TestSuite")

Uninstalling the Test Suite

There are two methods that can be used for uninstalling the Test Suite:

  • Uninstall the Test Suite using the Windows Control Panel.
    • For Windows XP, the Windows Control Panel "Add/Remove Programs" feature can be used.
      1. Open the Windows "Control Panel"
      2. Double-click "Add or Remove Programs".
      3. Highlight "SCORM 2004 4th Edition Test Suite Version 1.1.1".
      4. Click Remove and follow the instructions
    • For Windows Vista, use the Windows Control Panel "Programs and Features" feature.
      1. Open the Windows "Control Panel"
      2. Double-click "Programs and Features".
      3. Highlight "SCORM 2004 4th Edition Test Suite Version 1.1.1".
      4. Click Uninstall on the toolbar above the list and follow the instructions
  • If the Test Suite is already installed, running the install again will uninstall the software. This method will not re-install the Test Suite. You will need to run the install a third time to accomplish a re-installation.

Note: Test Suite may not appear in the "Add/Remove Programs" menu. If it does not appear, use the second method to uninstall.
