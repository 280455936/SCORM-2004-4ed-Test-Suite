Manifest Utility Test


The Manifest Utility Test verifies that the IMS Manifest complies with the rules specified in the SCORM 2004 4th Edition Content Package Application Profiles.

The purpose of this test is to verify that the IMS Manifest test subject:
  • Is named imsmanifest.xml
  • Is located at the root of the content package.
  • Is well-formed.
  • Validates against the IMS Content Packaging XML Schema Definition (XSD) Version 1.1.4 ( imscp_v1p1.xsd found at http://www.imsglobal.org/ ).
  • Validates against the SCORM 2004 4th Edition Content Packaging Extensions XML XSD Version 2.0 ( adlcp_v1p3.xsd found at http://www.adlnet.gov/ ).
  • Implements all mandatory elements correctly.
  • Uses optional elements correctly if they are used.
  • Uses extensions elements and if they are used, they are valid to their respective controlling document.
  • Contains IMS Simple Sequencing Information, and if so, verifies that the sequencing information is valid according to the IMS Simple Sequencing XML Schema Definition (XSD) ( imsss_v1p0.xsd and supporting files found at http://www.imsglobal.org/ ).
  • Contains SCORM CAM Sequencing extensions, and if so, verifies that the sequencing extensions are valid according to the SCORM 2004 4th Edition Sequencing Extensions XML XSD Version 2.0 ( adlseq_v1p3.xsd found at http://www.adlnet.gov/ ).
  • Contains SCORM CAM Navigation extensions, and if so, verifies that the navigation extensions are valid according to the SCORM 2004 4th Edition Navigation Extensions XML XSD Version 1.0 ( adlnav_v1p3.xsd found at http://www.adlnet.gov/ ).
