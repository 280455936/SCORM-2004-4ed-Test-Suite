Running the Test Suite - Introduction

To run the Test Suite, open the < install_directory>\TestSuite\main.htm" file in Internet Explorer, or use the shortcuts added to the desktop during the installation. Select the test you want to perform from the main menu.

Click the "About the ADL SCORM 2004 4th Ed. Conformance Test Suite" link to view information about how the Conformance Test Suite works, and how to operate the Test Suite.

Click the "View the ADL SCORM 2004 4th Ed. Conformance Matrix" link to view information about the Testing Requirements of each type of test component.

Each individual test offers step-by-step instructions, as well as an "About this Test" link that enables you to view detailed documentation about each test. The following section describes how to operate each test in detail.
