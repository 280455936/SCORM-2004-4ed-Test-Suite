The following is an excerpt from the ADL SCORM® Conformance Matrix that is covered by the Content Package Conformance Test:

Content Package
Conformance Label:
  • CP SCORM 2004 4th Edition Conformant

The Content Package shall adhere to the Testing Requirements defined for the following Conformance Categories:

  • Content Package Content Aggregation Model Version 1.1 (CP CAM 1.1)

  • Content Package Run-Time Environment Version 1.1 (CP RTE 1.1)
Conformance Category:
  • CP CAM 1.1
Requirements Summary
  • The content package shall conform with the requirements defined for the content package, and if the content package is a SCORM 2004 4th Edition Content Aggregation Content Package Application Profile
    • The manifest shall conform with the Content Aggregation Content Package Application Profile Manifest requirements.
    • If the manifest contains SCORM 2004 4th Edition sequencing information, then the sequencing extensions in the manifest shall conform with the SCORM 2004 4th Edition Sequencing Extension requirements.
    • If the manifest contains SCORM 2004 4th Edition navigation/presentation information, then the navigation/presentation extensions in the manifest shall conform with the SCORM 2004 4th Edition Navigation/Presentation Extension requirements.
  • If the content package is a SCORM 2004 4th Edition Resource Content Package Application Profile, then the manifest shall conform to the Resource Content Package Application Profile Manifest requirements, and
  • If the content package manifest contains metadata, then the metadata instance shall be well-formed and valid according to the corresponding provided Controlling Document (e.g., XSD, DTD).
Conformance Category:
  • CP RTE 1.1
Requirements Summary
  • The content package shall contain at least one SCO resource or asset resource.
  • All SCO resources identified in the manifest shall conform with the SCO Testing Requirements.
