About the ADL SCORM® 2004 4th Edition Test Suite Version 1.1.1

The ADL SCORM 2004 4th Edition Test Suite Version 1.1.1 provides the capability to assess conformance of learning management systems (LMS), Sharable Content Objects (SCOs) and content packages with the SCORM .

More information on the specific details of each individual test module is available by selecting any of the following links:

  • About the Learning Management System (LMS) Conformance Test
  • About the Content Package Conformance Test
  • About the Sharable Content Object (SCO) Run-Time Environment (RTE) Conformance Utility Test
  • About the Manifest Utility Test
  • About the Checksum Utility Test

For information regarding SCORM® , please visit  ADLNet.gov.

For a summary of the SCORM® Testing Requirements, please view the ADL SCORM® 2004 4th Edition Conformance Matrix.

Designed with a common look and feel, each test module includes step-by-step instructions, a test log, and a link to its About page. Testing activity can be monitored real-time while the test log is updated. The test tool provides the ability to print the log, save the log, or add textual comments to the log.
