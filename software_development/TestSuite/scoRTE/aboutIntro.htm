The following is an excerpt from the ADL SCORM® Conformance Matrix that is covered by the SCO Run-Time Environment Conformance Utility Test:

Sharable Content Object (SCO)
Conformance Label:
  • SCO SCORM 2004 4th Edition Conformant
The SCO shall adhere to the Testing Requirements defined for the following Conformance Categories:
  • SCO Run-Time Environment Version 1.1 (SCO RTE 1.1)

Conformance Category:
  • SCO RTE 1.1
Requirements Summary
  • The SCO shall search for and find an API Instance named API_1484_11 as a Document Object Model (DOM) object as defined in the SCORM 2004 4th Edition Run-Time Environment Version 1.1, and
  • The SCO shall successfully invoke, at a minimum, the Initialize("") and Terminate("" ) API methods, and
  • If used, the SCO shall successfully invoke the Data Transfer Methods, and
  • If used, the SCO shall successfully invoke the Support Methods, and
  • If using the Data Transfer Methods, the SCO shall ensure that all SCORM 2004 4th Edition Run-Time Environment Data Model elements used in the method calls adhere to the requirements of those elements.
